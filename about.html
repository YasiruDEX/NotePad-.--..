Keyboard Shortcuts

  • New File Ctrl+N / Cmd+N
  • Open File Ctrl+O / Cmd+O
  • Save File Ctrl+S / Cmd+S
  • Exit Ctrl+Q / Cmd+Q
  • Undo Ctrl+Z / Cmd+Z
  • Redo Ctrl+Y / Cmd+Shift+Z
  • Cut Ctrl+X / Cmd+X
  • Copy Ctrl+C / Cmd+C
  • Paste Ctrl+V / Cmd+V
  • Select All Ctrl+A / Cmd+A
  • Ask Gemini AI Ctrl+Shift+G / Cmd+Shift+G
  • Command Palette Ctrl+Shift+P / Cmd+Shift+P

Command Palette Commands

Press Ctrl+Shift+P (or Cmd+Shift+P on Mac) to access AI-powered commands:

  • Summarize Create a concise summary
  • Fix Grammar Correct grammatical errors
  • Convert to Bullet Points Format as bullet points
  • Expand Text Add detail and context
  • Simplify Text Make easier to understand
  • Explain Code Explain what code does
  • Fix Code Find and fix code issues
  • Translate to Spanish Translate text to Spanish

How to Use AI Feature

Direct AI Chat:
1. Set your Gemini API key in File → Preferences
2. Type your question or prompt in the editor
3. Press Ctrl+Shift+G (or Cmd+Shift+G on Mac)
4. The AI response will appear below your text


Command Palette:
1. Press Ctrl+Shift+P (or Cmd+Shift+P on Mac)
2. Select a command from the list (use arrow keys or click)
3. The AI will process your text and replace it with the result


System Instructions:
You can customize the AI's behavior by setting system-level instructions in File → Preferences. This is useful for setting a specific tone, role, or behavior across all AI interactions.

* On macOS, use Cmd instead of Ctrl
